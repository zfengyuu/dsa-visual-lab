# DSA Visual Lab

Interactive Data Structures and Algorithms learning app.

## MVP

- Web app first with Next.js, React, TypeScript, Tailwind CSS, Monaco Editor, and localStorage progress.
- Desktop app follows after the Web MVP is stable, using Tauri and the shared packages.

## Commands

```bash
pnpm install
pnpm dev
pnpm test
pnpm typecheck
pnpm build
```
